# Cloud Diligence Model

## Purpose

The cloud diligence model groups the major diligence domains into a repeatable assessment structure.

## Domains

- architecture
- reliability
- security and compliance
- disaster recovery
- FinOps
- modernization
- operating model

## How To Use

- score each domain separately
- capture evidence for the score
- note investment implications
- recommend remediation sequencing
- keep the scoring consistent across reviews
